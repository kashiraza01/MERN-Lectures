# Day 1: Web Fundamentals & Git Workflows

## Quick Start
1. Clone this repository: `git clone <repo-url>`
2. Open the folder in your code editor
3. Follow the **CLI Speed Drill** and **Conflict Exercise** in the lab sheet

## Lab Objectives
- Practice terminal navigation and file manipulation
- Understand the feature branch workflow
- Trigger, identify, and resolve a merge conflict
- Submit a clean Pull Request

## Need Help?
Run `git status` frequently. Check your branch with `git branch`. 
Ask an instructor before force-pushing or deleting remote branches.
